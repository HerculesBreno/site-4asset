## 🎯 Comandos disponíveis

| Comando | Quando usar |
|---|---|
| `/edit-copy [o que mudar]` | Edição direta — trocar texto, número, telefone, foto. Sem análise estratégica. **Use pra 80% dos pedidos.** |
| `/dream-team [pedido]` | Análise estratégica completa — mudar posicionamento, repensar headline, redesenhar funil. Convoca 9 especialistas. |
| `/seo-review [seção]` | Só Neil Patel + Rand Fishkin opinam sobre SEO de uma seção. |
| `/cro-review [seção]` | Só Peep Laja + Joanna Wiebe + Lenny Rachitsky opinam sobre conversão. |
| `/status` | Lista PRs abertos no repositório, com label `marketing-request`. |
| `/ajuda` | Mostra este menu. |

## 💬 Sem comando também funciona

Você pode escrever em **linguagem natural**, sem comando. Exemplos:

- "trocar o whatsapp do rodapé pra (48) 99999-8888" → vai pra `/edit-copy`
- "queria repensar o hero focando em transmissão de energia" → vai pra `/dream-team`
- "como estão os PRs abertos?" → vai pra `/status`

## 🚀 Como funciona depois que você aprova

1. Eu **abro um Pull Request** no GitHub com a mudança
2. Você (ou quem da liderança de marketing) **revisa o diff** no PR
3. Clica em **Merge** → site atualiza em ~30 segundos
4. Não rolou? **Reverter** é um click no GitHub
